# Talkie - 48h Hackathon - Dunarea de Jos University of Galati 

LIVE DEMO: [https://talkie.vercel.app](https://talkie-lucaay.vercel.app)

EN:
Social Media App

1. NodeJs must be installed.
2. Clone repo to VSCode.
3. Run "npm install"
4. Run "npm run dev".
5. Have fun!
